myIndihome

Application : myIndihome

Periode : July - Present

OverView

myIndihome is a mobile application owned by Telkom Indonesia. This application allows users to find out which internet package is being used, and for users who have not yet subscribed, they can order internet packages through this application.

What role do I play?

In this project I act as a UI Designer

Project Summary

In this project, I made improvements to local components according to the system design owned by Telkom Indonesia. After adjusting the local component with the system design owned by Telkom Indonesia, then I applied it to the existing design screen. During the process, I also play a role in setting the layout for each existing screen
