Galery

Landing Page

​​This is my first UI challenge in Binar Academy class, which is to make a landing page above the fold. In this challenge, I got inspiration from a bakery that is synonymous with the word sweet, so I gave it pink and white colors with an additional image of a macaroon in the middle to make the sweet impression more pronounced. in this challenge I get the first favorite rating

Countdown

This is a challenge to make a countdown screen user interface. In this challenge, I represent continuous time with an hourglass. Of course, there are many countdown screen references that still use a simple design, namely only using numbers that keep going backwards, therefore I represent it with something that is not boring, namely by adapting an hourglass.

Flash Screen

​​In this challenge I am required to create a bakery splash screen page. Here I use a brown background to match the logo of the bakery. In this challenge I also made a full prototype which you can see at the following link https://dribbble.com/ur_dsgnr

Pop Up

The fourth challenge is to make a popup screen. In this challenge, I made a popup about a discount on an application. I made this popup as attractive as possible with 3d illustrations and ornaments such as circular orbits on the popup card.

Leaderboard

​In this challenge leaderboard screen, I ranked the top 3 to catch users' attention. With the addition of an up or down icon it makes it easier for users to find out the progress of their rankings, besides that the scores displayed are also very helpful for users
